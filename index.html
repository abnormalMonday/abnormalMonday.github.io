CSS Basic Properties

This is my version of a CSS reference sheet for basic CSS properties.

Text Properties

Property Description Values
color Sets the foreground color ie text rgb, hsl, hex, keyword
line-height Sets vertical space around text normal, px, em, %
letter-spacing Adjust space between characters normal, px, em, %
text-align Aligns text to parent element left, right, center, justify
text-decoration Adds flare to characters none, underline, overline, line-through
text-indent Indents the first line of text in an element length, %
text-transform Controls the characters within an element none, capitalize, uppercase, lowercase
